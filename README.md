# MediaVault

Aplicação de catálogo de filmes feita com **Flask** e **SQLite**.

## 🚀 Como rodar localmente (Linux)

1. Crie um ambiente virtual:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

3. Execute o app:
   ```bash
   python app.py
   ```

4. Abra no navegador:
   ```
   http://127.0.0.1:5000
   ```
